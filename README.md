# BOT AMBIENT

![Fitur Ambient](Ambient.png)

**Fitur:**

**• Auto Chat**

**• Multi Account**

**• Supports Proxy w/o Proxy**

## Requirements

Before running this project, make sure you have installed:

- Node.js
- npm (Node Package Manager)

## Installation

1. **Clone this repository:**

    ```plaintext
    git clone https://github.com/TahuriGroup/AMBIENT
    cd AMBIENT

2. **Install the required packages:**

    ```plaintext
    npm install fs axios crypto https-proxy-agent socks-proxy-agent querystring

3. **Add your accounts in the account.txt file. Example:**

    ```plaintext
   email@gmail.com|12345
    email@gmail.com|12345
    email@gmail.com|12345

4. **If you want to use a proxy, edit the proxy.txt file. Example:**
   ```plaintext
   http://username:password@ip:port

5. **Run the script:**
   ```plaintext
   node main.js
